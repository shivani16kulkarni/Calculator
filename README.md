# Calculator
This is a basic calculator web app that allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division. Built using HTML, CSS, and JavaScript, it provides a clean and responsive interface for basic calculations.

---

## About the Project
This is a front-end practice project created to build and style a functional calculator using only HTML, CSS, and JavaScript. It focuses on DOM manipulation, event handling, and responsive layout without the use of any frameworks or libraries.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
---
## Concepts Practiced
- Used semantic HTML to build calculator structure (buttons, display screen)

- Handled click events for each button using JavaScript

- Used string evaluation to compute arithmetic expressions

- Implemented a responsive layout using Flexbox

- Styled the interface to be clean, simple, and user-friendly

- Applied basic validation to avoid invalid input sequences
